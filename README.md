### Jana

AI Assistant for the Frappe Ecosystem

Jana is an AI-powered assistant that integrates into Frappe Desk, providing context-aware chat, document understanding, and natural language interaction across any Frappe-based application.

### Features

- AI chat widget on every Frappe Desk page
- Context-aware — knows which document you're viewing
- BYOK (Bring Your Own Key) — use OpenAI, Anthropic, Google, Ollama, or any compatible provider
- Agent framework with customizable system prompts and tools
- Template marketplace for sharing agent configurations

### Installation

```bash
cd $PATH_TO_YOUR_BENCH
bench get-app https://github.com/tonic-6101/jana.git
bench --site your-site install-app jana
```
